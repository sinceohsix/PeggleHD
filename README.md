### Features

- Updated UI, Backgrounds, Characters, and more!
- Pixel-perfect HD textures ported straight from the PS3.
- Texture adjustments and fixes that were originally missing.
- Multiple variants supporting new controllers and layouts.
- <s>KTX2 compression for fast loading, wide support, and small file size.</s> (Coming soon!)
- HD (720p) Peggle action in your pocket!

### Installation
Extract your chosen release and drag the `NPUG30040` folder to the `.../PSP/TEXTURES` directory on your device. This location is slightly different depending on the device.

- Nintendo Switch  
```
/switch/ppsspp/config/ppsspp/PSP/TEXTURES
```

- macOS (from your home folder)  
```
.config/ppsspp/PSP/TEXTURES
```

- iOS   
```
On My iPhone/iPad > PPSSPP > PSP > TEXTURES
```

If your device/operating system is not listed, the pack still works (I just don't have said device and don't want to give improper directions). Try opening the downloaded `NPUG30040` as a game in PPSSPP and it should install itself.
